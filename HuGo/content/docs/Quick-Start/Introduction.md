# ClickHouse 简介

- 开源的 **OLAP** 数据库管理系统
- **列式存储**，允许通过 **SQL** **实时**生成分析报告
- **每台服务器 每秒处理数亿行数据**
- 与 GreenPlum、Vertica 等对比数据： https://clickhouse.tech/benchmark/dbms/
- 可水平扩展



## 适用场景

- APM 分析
- 监控数据
- 时序数据
- ...

## 不适用场景

- OLTP
- Blob 或 文档
- 高并发
